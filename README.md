# Dot Files

These are config files to set up a system using [Oh My ZSH](https://github.com/robbyrussell/oh-my-zsh) on MacOS and Linux.

## Installation

Install dotfiles

```terminal
git clone https://github.com/jisraelsen/dotfiles.git ~/.dotfiles
```

Symlink dotfiles

```terminal
~/.dotfiles/script/setup
```
